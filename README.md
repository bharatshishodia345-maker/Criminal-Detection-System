# Criminal Detection System

An AI-powered Criminal Detection and Information Management System developed using Python, OpenCV, Flask, and Machine Learning.

## Features

- Real-Time Face Detection
- Criminal Face Recognition
- Unknown Person Detection
- Evidence Image Storage
- Alert & Siren System
- Criminal Database Management
- Admin Login Dashboard
- Access Logs Monitoring
- Model Retraining Support

## Technologies Used

- Python
- OpenCV
- Flask
- HTML/CSS
- Machine Learning (LBPH Face Recognition)
